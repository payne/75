Notes: Cable loss coefficients are approximate and representative of typical cable; actual loss varies by manufacturer. Dipole and inverted-V impedances are interpolated from NEC-modeled data over average ground (σ=5 mS/m, εr=13). The inverted-V 90° model uses 45° droop each side; the 120° model uses 30° droop each side. Off-resonance reactance uses a series-RLC approximation with the stated Q factor. Shack-end SWR is computed using the full transmission line equation (Zin = Z₀ · (ZL + Z₀·tanh(γl)) / (Z₀ + ZL·tanh(γl))), then the reflection coefficient is referenced to a 50Ω source impedance. This correctly accounts for impedance transformation along the line, including cases where the line Z₀ differs from the source impedance. These models are adequate for comparison purposes but are not a substitute for measured data or modeling your specific antenna and site.
